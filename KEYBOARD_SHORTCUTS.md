# Keyboard Shortcuts Reference

This document outlines the standard keyboard shortcuts implemented in Project 042-X to ensure a fast, keyboard-first developer workflow.

## Global Navigation
| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl/Cmd + K` | **Command Palette** | Opens the global command palette for actions and commands. |
| `Ctrl/Cmd + P` | **Quick Open** | Opens fuzzy file search to jump to any file instantly. |
| `Esc` | **Close/Cancel** | Closes palettes, context menus, or clears focus. |

## Tabs & Editor
| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl/Cmd + W` | **Close Tab** | Closes the currently active code tab. |
| `Ctrl/Cmd + Shift + T`| **Reopen Tab** | Reopens the last closed tab (Future enhancement). |
| `Ctrl/Cmd + \`` | **Toggle Sidebar** | Hides or shows the left Explorer sidebar. |

## Graph Controls (When active)
| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `+ / -` | **Zoom** | Zooms the graph in and out. |
| `Space + Drag` | **Pan** | Pans the graph canvas. |
| `F` | **Fit to View**| Fits all nodes into the current viewport. |
